NAME
ALGORITHM - graph traversal DFS or DFS algorithm

SYNOPSIS
ALGORITHM -h
ALGORITHM -d graphname
ALGORITHM -b graphname
ALGORITHM -s graphname
ALGORITHM -g graphname

DESCRIPTION
ALGORITHM traverses the current graph with help Depth-First Search and Breadth-First Search algorithmes and create matrix shortest path
Example of DFS:
A -> B -> D -> C
Example of BFS:
A -> C -> B -> D
Example of matrix shortest path:
0 2 1
4 0 0
3 0 0

COMMAND-LINE OPTIONS
-h, --help - option that shows this text
-d, --dfs - algorithm DFS for graph
-b, --bfs - algorithm BFS for graph
-s, --shortest_path - create matrix shortest path with help algorithm WFI
-g, --hamilton - output hamilton`s cycles
