NAME
VIEW - find vertex degree and component connections

SYNOPSIS
VIEW [-h]
VIEW -d graphname
VIEW -c graphname
VIEW -a

DESCRIPTION
VIEW this command find vertex degree and component connections the graph's
Example of vertex degree:
A: 1
B: 2
C: 3
Example of component connections: in current graph has 2 connections componentCOMMAND-LINE OPTIONS
-h, --help - option that shows this text
-d, --vertex_degree - find vertex degree
-c, --connected-components - find connected components
-a, --all - find vertex degree and connected components
